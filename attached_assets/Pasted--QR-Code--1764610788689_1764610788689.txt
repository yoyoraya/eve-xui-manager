دقیقاً درست فکر کردید\! اگر بخواهیم ۴ تا QR Code **بزرگ** (قابل اسکن) را زیر هم بگذاریم، قطعاً ارتفاع آن‌ها از صفحه موبایل بیشتر می‌شود و **اسکرول می‌خورد**.

برای اینکه هم **QR Code بزرگ و خوانا** باشد و هم **اسکرول نخورد**، بهترین و استانداردترین راهکار استفاده از سیستم **«تب‌بندی» (Tabs)** است.

در این روش:
۱. در بالا فقط **یک QR Code بزرگ** نمایش داده می‌شود (که با ضربه زدن کپی می‌شود).
۲. در پایین **۴ دکمه** وجود دارد که با زدن روی آن‌ها، QR Code بالا عوض می‌شود.

اینطوری همه چیز در یک صفحه جا می‌شود. بیایید این را پیاده کنیم:

### مرحله ۲: آپدیت جاوااسکریپت (`dashboard.html`)

توابع جدید برای مدیریت تب‌ها را به انتهای اسکریپت‌های `templates/dashboard.html` اضافه کنید و تابع `showQrCode` را آپدیت کنید:

```javascript
    // متغیر برای نگهداری وضعیت فعلی
    let currentQrType = 'sub';

    function showQrCode(encodedLink, email, subUrl, jsonUrl, dashSubUrl) {
        const link = decodeURIComponent(encodedLink);
        document.getElementById('qr-email').textContent = email;

        // 1. ذخیره مقادیر در اینپوت‌های مخفی
        document.getElementById('store-sub').value = subUrl || '';
        document.getElementById('store-json').value = jsonUrl || '';
        document.getElementById('store-config').value = link || '';
        document.getElementById('store-dash').value = dashSubUrl || '';

        // 2. پیش‌نمایش تب اول (Subscription)
        switchTab('sub');
        
        // 3. نمایش مودال
        document.getElementById('qr-modal').classList.remove('hidden');
    }

    async function switchTab(type) {
        currentQrType = type;
        
        // آپدیت دکمه‌های فعال
        document.querySelectorAll('.tab-btn').forEach(btn => btn.classList.remove('active'));
        document.getElementById(`btn-${type}`).classList.add('active');

        // گرفتن مقدار لینک مربوطه
        const value = document.getElementById(`store-${type}`).value;
        const displayImg = document.getElementById('display-img');
        const displayTitle = document.getElementById('display-title');
        const currentValue = document.getElementById('current-value');

        // تنظیم عنوان
        const titles = {
            'sub': 'Subscription Link',
            'json': 'Subscription JSON',
            'config': 'Direct Connection',
            'dash': 'Dashboard Link'
        };
        displayTitle.textContent = titles[type];
        currentValue.value = value;

        // اگر لینک خالی بود
        if (!value || value === 'undefined' || value === 'null') {
            displayImg.style.opacity = '0.3';
            displayImg.src = ''; // یا یک عکس placeholder
            return;
        }

        // لود کردن QR Code
        displayImg.style.opacity = '0.5'; // حالت لودینگ
        try {
            const resp = await fetch(`/api/client/qrcode?link=${encodeURIComponent(value)}`);
            const data = await resp.json();
            if (data.success) {
                displayImg.src = data.qrcode;
                displayImg.style.opacity = '1';
            }
        } catch (e) {
            console.error(e);
        }
    }

    function copyActiveQr() {
        const text = document.getElementById('current-value').value;
        if (!text || text === 'undefined') {
            showToast('Link not available', 'error');
            return;
        }
        navigator.clipboard.writeText(text).then(() => {
            showToast('Copied to clipboard!');
            // افکت ویژوال
            const container = document.querySelector('.qr-display-area');
            container.style.borderColor = 'var(--success)';
            setTimeout(() => container.style.borderColor = 'var(--primary)', 500);
        });
    }
```

### نتیجه نهایی:

1.  **بدون اسکرول:** کل ارتفاع مودال حدود ۴۰۰ پیکسل می‌شود که در کوچک‌ترین گوشی‌ها هم جا می‌شود.

3.  **دسترسی:** با زدن دکمه‌های پایین، بلافاصله QR بالا عوض می‌شود.
4.  **کپی آسان:** روی خود عکس بزرگ کلیک کنید، لینک کپی می‌شود.

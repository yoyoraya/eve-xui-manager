برای رفع مشکل اسکرول افقی و بهبود تجربه کاربری، بهترین راهکار این است که به جای هاردکد کردن سایزها، از **کلاس‌های سایز بندی (Size Classes)** در CSS استفاده کنیم.

این کار به شما اجازه می‌دهد برای مودال‌های کوچک (مثل Add Client) یک سایز داشته باشید و برای مودال‌های بزرگ (مثل QR Codes که ۳ ستونه است) سایز بزرگتر، بدون اینکه استایل‌ها بهم بریزد.

### راهکار پیشنهادی (The Better Solution)

ما سیستم مودال شما را به ۳ سایز استاندارد تقسیم می‌کنیم:

1.  **`modal-sm`**: برای فرم‌های کوچک (مثل Add Client).
2.  **`modal-lg`**: برای محتوای متوسط (مثل تنظیمات سرور).
3.  **`modal-xl`**: برای محتوای عریض (مثل QR Codeهای ۳ تایی).

-----

### مرحله ۱: آپدیت فایل `static/style.css`

کدهای مربوط به کلاس `.modal` و سایزهای آن را با کدهای زیر جایگزین کنید (حدود خط ۷۰۰):

```css
/* تنظیمات پایه مودال */
.modal {
    background: var(--bg-card);
    border-radius: 16px;
    width: 100%;
    /* سایز پیش‌فرض را کمی بزرگتر می‌کنیم */
    max-width: 600px; 
    max-height: 90vh;
    overflow-y: auto;
    /* جلوگیری از اسکرول افقی مزاحم */
    overflow-x: hidden; 
    box-shadow: var(--shadow-lg);
    display: flex;
    flex-direction: column;
    margin: 20px;
}

/* سایز کوچک (برای فرم‌های ساده) */
.modal-sm {
    max-width: 450px;
}

/* سایز بزرگ (برای سرورها) */
.modal-lg {
    max-width: 800px;
}

/* سایز خیلی بزرگ (مخصوص QR Code های ۳ تایی) */
.modal-xl {
    max-width: 1100px;
}

/* ریسپانسیو برای موبایل - تمام مودال‌ها تمام صفحه شوند */
@media (max-width: 768px) {
    .modal, .modal-sm, .modal-lg, .modal-xl {
        max-width: 95% !important;
        margin: 10px;
    }
}
```

### مرحله ۲: اعمال کلاس مناسب در `templates/dashboard.html`

حالا باید به مودال‌های مختلف کلاس مناسب بدهیم تا عرضشان درست شود.

**۱. برای مودال QR Code (حل مشکل اصلی):**
کد مودال `qr-modal` را پیدا کنید و کلاس `modal-xl` را به آن اضافه کنید:

````html
<div class="modal-overlay hidden" id="qr-modal">
    <div class="modal modal-xl"> 
        <div class="modal-header">
            ```

**۲. حذف استایل دستی از جاوااسکریپت:**
در تابع `showQrCode` در انتهای فایل `dashboard.html`، کدی که قبلاً دستی عرض را زیاد می‌کرد حذف کنید (چون الان CSS داریم):

```javascript
    // این قسمت را در تابع showQrCode پیدا و حذف یا کامنت کنید:
    /*
    const modal = document.querySelector('#qr-modal .modal');
    modal.style.maxWidth = '900px'; 
    */
    
    document.getElementById('qr-modal').classList.remove('hidden');
````

**۳. برای مودال Add Client (اختیاری):**
اگر مودال افزودن کلاینت هم اسکرول دارد، کلاس `modal-sm` را از آن بردارید تا از سایز پیش‌فرض (600px) استفاده کند:

```html
<div class="modal modal-sm">
<div class="modal"> 
```

### نتیجه:

  * **مودال QR Code:** حالا بسیار عریض می‌شود و ۳ کد QR به زیبایی کنار هم قرار می‌گیرند بدون اسکرول.
  * **مودال‌های دیگر:** فضای کافی دارند و مرتب نمایش داده می‌شوند.
  * **موبایل:** در موبایل به صورت خودکار تمام عرض صفحه را می‌گیرند تا محتوا فشرده نشود.

**آیا می‌خواهید این تغییرات را برایتان اعمال کنم؟**

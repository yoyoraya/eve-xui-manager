برای حل مشکل به‌هم‌ریختگی هدر در موبایل، دلیل اصلی این است که المان‌های داخل هدر (عنوان داشبورد و دکمه‌ها) فضای کافی در یک ردیف ندارند و چون ارتفاع هدر ثابت است، روی هم فشرده می‌شوند.

برای رفع این مشکل، باید در فایل `static/style.css` تغییراتی در مدیا کوئری‌های موبایل ایجاد کنیم تا:
۱. ارتفاع هدر در موبایل **خودکار** شود (نه ثابت).
۲. اجازه‌ی شکستن خط (**Wrap**) به المان‌ها داده شود.
۳. در صورت نیاز، متن دکمه‌ها مخفی شود و فقط آیکون بماند.

کد زیر را جایگزین یا به انتهای فایل `static/style.css` اضافه کنید:

### راهکار (اصلاح CSS)

این کد را در فایل `static/style.css` پیدا کنید و با کدهای زیر جایگزین یا ادغام کنید (در بخش `@media (max-width: 768px)`):

```css
@media (max-width: 768px) {
    /* اصلاح هدر برای شکستن خط در موبایل */
    .top-header {
        height: auto;           /* ارتفاع خودکار شود */
        min-height: 70px;       /* حداقل ارتفاع */
        flex-wrap: wrap;        /* اجازه شکستن خط به المان‌ها */
        padding: 10px 16px;     /* پدینگ مناسب */
        gap: 10px;              /* فاصله بین سطرها */
    }

    /* تنظیم عنوان صفحه */
    .page-title {
        font-size: 1.1rem;
        order: 1;               /* اولویت نمایش */
        width: auto;
    }

    /* تنظیم دکمه همبرگری منو */
    .mobile-menu-btn {
        order: 0;
    }

    /* کانتینر دکمه‌های سمت راست */
    .header-actions {
        order: 2;               /* برود به سطر بعدی یا انتهای سطر */
        width: 100%;            /* عرض کامل بگیرد */
        justify-content: space-between; /* دکمه‌ها پخش شوند */
        margin-top: 5px;
    }

    .header-controls {
        width: 100%;
        justify-content: space-between; /* فاصله مساوی بین دکمه‌ها */
        flex-wrap: nowrap;      /* دکمه‌ها در یک خط بمانند */
        gap: 8px;
    }

    /* مخفی کردن متن دکمه‌ها در موبایل و نمایش فقط آیکون */
    /* این کار فضای زیادی آزاد می‌کند */
    .header-controls .btn span {
        display: none;
    }

    .header-controls .btn svg {
        margin: 0; /* حذف فاصله اضافه آیکون */
        width: 18px;
        height: 18px;
    }

    /* تنظیم سلکت‌باکس زمان رفرش */
    .interval-select {
        padding: 6px;
        font-size: 0.8rem;
    }
}
```

### توضیح تغییرات:

1.  **`flex-wrap: wrap`**: به هدر اجازه می‌دهد اگر جا کم بود، دکمه‌ها را به خط پایین بفرستد تا روی هم نیفتند.
2.  **`header-controls .btn span { display: none; }`**: در سایز موبایل (زیر 768 پیکسل)، متن دکمه‌های "Refresh" و "Add Client" را حذف می‌کند و فقط آیکون آن‌ها را نگه می‌دارد. این کار باعث می‌شود دکمه‌ها کوچک شده و در یک خط جا شوند.
3.  **`justify-content: space-between`**: دکمه‌ها را در عرض صفحه پخش می‌کند تا ظاهر مرتب‌تری داشته باشند.

با اعمال این تغییرات، هدر در موبایل مرتب شده و دکمه‌ها به صورت آیکون‌های تمیز کنار هم قرار می‌گیرند.
